## 弧光追踪器（悲剧适用）

### 弧光5阶段

| 阶段 | 信念状态 | 表现 |
|------|----------|------|
| 1. 固守 | "只要我足够好，就会被看见" | 骄傲、拼命证明自己 |
| 2. 裂缝 | "我做了为什么还是不行？" | 第一次被无视/打压后的困惑 |
| 3. 动摇 | "也许我就是不配" | 反复受挫后开始自我怀疑 |
| 4. 接受 | "我不配，但我不想认" | 知道真相但仍挣扎 |
| 5. 沉默 | "算了" | 不再挣扎，不是释然是放弃 |

### 弧光铁律

1. 弧光必须逐集递进，不能跳步
2. 弧光可以暂停但不能倒退
3. 弧光通过"选择"完成，不是台词
4. 配角弧光3阶段即可
5. 弧光阶段和节拍表联动：裂缝→催化事件，动摇→争论/中点，接受→一无所有

## 单集节奏公式（60-120秒/集）

```
0-3秒   → 抓人：爆发痛点/最强矛盾/上集钩子回应
3-10秒  → 冲刺：进入核心矛盾，不给喘息
10-30秒 → 叠压：层层叠加矛盾，每10秒一个新信息
30秒处  → 转折：必须出现一次转折/新信息/视角切换
30-90秒 → 闭环：让至少一个因果链闭合
最后3-5秒 → 钩子：致命断点/追更钩子
```

## 输出格式

```
【全剧节拍表】
按15节拍格式，填写每节拍的集数映射和具体内容

【23段落大纲】
按23段落格式，填写每段落的集数范围和核心任务

【弧光追踪表】
| 集数 | 主角弧光 | 配角A弧光 | 配角B弧光 |
按集数分配弧光阶段

【单集节奏设计】
选3个关键集，输出该集的0-3秒-10秒-30秒-结尾钩子
```

## 任务

1. 输出完整的救猫咪15节拍表（映射到{total_episodes}集）
2. 输出23段落大纲
3. 输出弧光追踪表（悲剧用5阶段，其他类型用简化版）
4. 选取3个关键集，输出单集节奏设计

## 类型专项适配

- **悲剧**：使用下沉阶梯节奏+微光碾碎节点
- **甜宠**：使用甜虐交替节奏+双向奔赴弧光
- **悬疑**：使用信息差递进节奏+反转节点
- **非遗**：传承线与情感线双轨并行
